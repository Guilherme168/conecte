"Conecte" é o nome da aplicação desenvolvida como conclusão de trabalho final para a Atividade Extensionista II. 
Seu intuito é guiar o público de mais idade e/ou com pouca familiaridade com tecnologia aos serviços oficiais de maior interesse para o público, como serviços governamentais e suas vertentes, tudo de forma segura e direta.
O objetivo aqui NÃO é substituir, ou ofereçer, algum tipo de serviço oficial, e sim manter o usuário seguro e protegido durante sua navegação até o seu destino.
